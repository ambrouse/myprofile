# Ban can gui gi cho minh (rat don gian)

## 1) Anh tham chieu (khi ban san sang)
Gui 5-10 anh theo gu ban thich:
- 2-3 anh tong the scene (vibe cyberpunk/tech).
- 2-3 anh chat lieu (kim loai, panel, anh sang).
- 1-2 anh UI/HUD ban thay dep.
- 1-2 anh ban KHONG thich (de tranh lam sai huong).

## 2) Model 3D (co the gui dan)
Chi can copy file `.glb` vao thu muc `assets3d/` dung ten:
- `ai_core_reactor.glb`
- `skills_node.glb`
- `rag_data_vault.glb`
- `edge_inference_bay.glb`
- `comms_terminal.glb`

## 3) Neu chua co model
Khong sao. Web van chay fallback scene hien tai.
Mình se tiep tuc toi uu scene va doi model tung phan khi ban gui them.

## 4) Khi ban da copy model
- Chay: `python -m http.server 8000`
- Mo: `http://localhost:8000`
- Hard refresh: `Ctrl + F5`

HUD se hien `ASSETS x/5` de ban biet model da nap duoc bao nhieu.
