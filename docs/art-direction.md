# Art Direction - AI Command Nexus

## Vision
- The portfolio lives inside a futuristic command citadel where data, models, and orchestration systems are physically represented as architecture.
- Tone is dark, high-contrast, industrial sci-fi with controlled neon accents.

## Visual Pillars
- Scale first: foreground, midground, background must be clearly layered.
- Material clarity: brushed metal, matte carbon, holographic glass.
- Light storytelling: key/fill/rim with moving accents, not flat ambient-only lighting.
- Atmosphere: subtle fog and depth cues to avoid flat scene perception.

## Color Script
- Base dark: `#05070D`, `#0A1020`, `#111A2D`
- Primary tech blue: `#2F7CFF`, `#55A8FF`
- AI cyan accent: `#72E8FF`, `#6AF7DB`
- Secondary electric violet (limited): `#6D63FF`

## Environment Language
- Central AI Core: symbolic heart of the portfolio.
- Data Platforms: circular, layered, connected by bridges.
- Signal Pillars: repeated vertical structures for rhythm and scale.
- Hologram Consoles: screens for section context anchors.

## Motion Language
- Camera has mass (spring damped), no robotic snapping.
- Scene elements have slow autonomous motion loops.
- Intro uses cinematic fly-in, then shifts to realtime interactive mode.

## UI Integration Rules
- DOM UI stays readable and responsive on top of scene.
- 3D is world context, UI is control/hud layer.
- Never let postFX reduce text contrast below accessibility threshold.
